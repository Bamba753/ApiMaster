# ApiMaster: API Testing Tool
ApiMaster is a simple Qt-based application designed to help you test and interact with RESTful APIs. It allows users to send HTTP requests (GET, POST, PUT, DELETE), view JSON responses, and debug APIs.

Features
Send HTTP GET, POST, PUT, DELETE requests.

View and format JSON responses.

Add custom HTTP headers.

Handle and display error messages.

Save the results for future reference.
